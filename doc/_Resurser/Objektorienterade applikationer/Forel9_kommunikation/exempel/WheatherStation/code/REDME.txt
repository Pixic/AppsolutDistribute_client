Kompilering och exekvering i DOS
================================

rotkatalog: weatherstation

Kompilering
-----------

> cd display_client
> javac *.java

> cd sensor_client
> javac -sourcepath .. *.java

> cd weather_server
> javac -sourcepath .. *.java

Exekvering
----------
Starta *först* servern i en egen konsol. 
Därefter några olika klienter i var sin konsol.

> cd weatherstation

Starta server:

> java weather_server.Main <sensorport> <displayport>

Ex.
sensorport:	1234
displayport: 	1235


Starta sensorklient:

> java sensor_client.Main <serveraddress> <sensorport>

Ex.
serveraddress:	127.0.0.1
sensorport:	1234


Starta displayklient:

> java display_client.Main <serveraddress> <displayport>

Ex.
serveraddress:	127.0.0.1
sensorport:	1235
