Kör i DOS-prompt eller Linux-shell:

- en prompt för applikationen
- arbeta med komponenterna i en annan prompt, samtidigt som applikationen
- körs. 
- lägg till och kompilera en ny kommandoklass i components
- testa det nya kommandot i applikationsfönstret - ingen omkompilering behövdes! 

Kompilering: se resp. README i subkatalogerna

Exekvering:

> cd command_factory

> java application.Main components
